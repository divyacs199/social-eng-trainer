# AI Social Engineering Trainer

## 🔗 Live Demo
 https://social-eng-trainer.preview.emergentagent.com/

## 📌 About
This project simulates phishing attacks (Email, SMS, URL) to train users against social engineering threats.

## ⚠️ Note
The live demo was created using an AI-based app builder (Emergent).

This repository contains a manually recreated version using:
- HTML
- CSS
- JavaScript

## 🚀 Features
- Generate phishing simulations
- Multiple attack types
- Difficulty levels
